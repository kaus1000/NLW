## :bookmark: Resumo
<br />

 
Projeto que consiste no ranqueamento de perguntas para que as perguntas prioritárias dos aprensetadores
das lives sejão as mais ranqueadas ou seja, as dúvidas que são mais frequentes pela comunidade.
